# Sweet Shop Management System (Java + React)

## Setup Instructions

1. **Backend**:
   - `cd backend`
   - `mvn spring-boot:run`
   - API runs on: http://localhost:8080

2. **Frontend**:
   - `cd frontend`
   - `npm install`
   - `npm run dev`
   - UI runs on: http://localhost:5173

3. **AI Usage**:
   - Used Gemini to generate project structure and JWT logic.
